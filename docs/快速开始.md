# 快速开始

## 起步
### 克隆项目
```shell
git clone https://github.com/Barrierml/Naruse.git
```
### 安装依赖
```shell
npm install
# or
cnpm install
# or 
yarn
```
### 预览
```shell
npm run start
# 输出
# dist/naruse.dev.debug.js    编译后的代码
# dist/naruse.dev.debug.json  爱用测试专用
# dist/naruse.min.js					压缩后的代码

# PC端查看效果
cd ./h5-demo
npm run dev

# 小程序端查看效果
# 直接使用淘宝小程序开发者工具打开Naurse根目录即可
```
### 配置自己的组件
```js
// naruse.config.js
module.exports = {
    ...
    input: './src/adverts/index', // 修改为自己组件路径
    ...
};
```
